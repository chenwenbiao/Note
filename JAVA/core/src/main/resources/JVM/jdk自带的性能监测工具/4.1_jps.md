jps(Java Virtual Machine Process Status Tool)是JDK 1.5提供的一个显示当前所有java进程pid的命令

```shell
jps

# -l 输出应用程序main class的完整package名 或者 应用程序的jar文件完整路径名
jps -l

# -v 输出传递给JVM的参数
jps -v

# -m 输出传递给main 方法的参数
jps -m

# 这些命令也可以联合使用
jps -lmv

# 查看虚拟机的参数
jcmd pid VM.flags
```

>注：jps命令有个地方很不好，似乎只能显示当前用户的java进程，要显示其他用户的还是只能用unix/linux的ps命令。
