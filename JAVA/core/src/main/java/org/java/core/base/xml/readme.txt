XML是广泛使用的用于存储或传输数据技术，并且它与平台无关。 

Java提供各种API来读取，写入或操作XML数据。 

一些常用的java xml解析器是:

DOM Parser
SAX Parser
StAX Parser
JAXB

还有一些其他API也可用于java中的XML解析，例如JDOM和JiBX。

Java XML Parser - DOM

DOM Parser是最容易学习的java xml解析器。 
DOM解析器将XML文件加载到内存中，我们可以逐节点遍历它来解析XML。 
DOM Parser适用于小文件，但是当文件大小增加时，它执行速度慢并消耗更多内存。
