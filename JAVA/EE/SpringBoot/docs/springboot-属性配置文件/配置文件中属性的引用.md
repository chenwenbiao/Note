# 属性的引用

    可以在配置文件中引用前面已经配置了的属性：${app.name:默认值}

```properties
com.blog.name=name
com.blog.title=title
com.blog.desc=${com.blog.name} - 《${com.blog.title}》
```

~~~yaml
person:
  name: "张三\n李四${random.int}"
  age: 14
  map: {k1: V1,k2: V2}
  list:
    - cat
    - dog
    - pig
  cat:
    name: '${person.name}小猫\n小狗'
    age: 4
~~~
