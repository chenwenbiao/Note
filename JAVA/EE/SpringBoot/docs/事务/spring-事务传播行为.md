# 什么是事务的传播行为

    一个方法运行在了一个开启了事务的方法中时，当前方法是使用原来的事务还是开启一个新的事务
    这个就是事务的传播行为

![](../pics/事务的传播行为定义.png)

# 事务的传播行为如何设置呢?

    事务的传播行为 使用@Transactional中的一个属性叫Propagation来设置.
