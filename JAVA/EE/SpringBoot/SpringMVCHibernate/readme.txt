## Spring MVC Hibernate MySQL集成CRUD示例教程
 
我们在上一篇教程中学习了如何集成Spring和Hibernate。 今天将学习Spring MVC和Hibernate框架在Web应用程序CRUD示例中。

### Spring MVC Hibernate示例

请注意，我使用的是Spring 4.0.3.Release和Hibernate 4.3.5.Final版本，同样的程序也兼容Spring 4和Hibernate 3，但是你需要在spring bean 配置文件中进行一个小的更改,这个更改已经在上一个教程讨论。
